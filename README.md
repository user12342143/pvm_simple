# PVM Simple

A Payday 2 viewmodel modification mod that provides intuitive in-game customization of weapon viewmodel positioning and visual adjustments.

## Overview

PVM Simple allows players to customize how their weapons appear in the first-person view without complex external tools or configuration files. All adjustments can be made directly in-game through user-friendly sliders, making it easy to find your perfect viewmodel setup.

## Features

### Position Adjustments
- **Left/Right** - Move your viewmodel left or right on the screen
- **Forward/Backward** - Adjust the distance of the weapon from the camera
- **Up/Down** - Vertical positioning of the viewmodel

### Rotation Adjustments
- **Yaw** - Rotate the weapon left and right (around vertical axis)
- **Pitch** - Rotate the weapon up and down (around horizontal axis)
- **Roll** - Rotate the weapon clockwise and counterclockwise (around depth axis)

### Additional Visual Adjustments
- Real-time preview of all changes
- Easy-to-use in-game slider interface
- Save and load custom presets
- Restore default positions with a single click

## Installation

1. Download the latest release from the Releases section
2. Extract the files to your Payday 2 mods directory
3. Launch Payday 2 and enable the mod in the mod menu
4. Access PVM Simple settings during gameplay

## Usage

1. **In-Game Access** - Open the mod menu during a heist or in the main menu
2. **Adjust Sliders** - Use the intuitive sliders to customize your viewmodel position and rotation
3. **Preview Changes** - See changes in real-time as you adjust each value
4. **Save Your Setup** - Save your favorite configurations for future use
5. **Load Presets** - Quickly switch between saved viewmodel presets

## Configuration

All viewmodel adjustments are saved automatically and will persist between sessions. You can also manually edit configuration files if you prefer precise numeric control.

### Slider Ranges

- **Position (X/Y/Z)**: -100 to 100
- **Rotation (Yaw/Pitch/Roll)**: -180 to 180 degrees
- **Scale**: 0.5 to 2.0x
